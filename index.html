1."We are interested in retrieving only the students' names; all the names should be in caps.
['JOHN', 'BABA', 'YAGA', 'WICK']"

Ans: let studentNamesInCaps = studentRecords.map(student => student.name.toUpperCase());
console.log(studentNamesInCaps);
The map function takes an array and applies a callback function to each element in the array
and return the output


2."Suppose we have the same dataset as above but this time we want to get the details of students who scored more than 50 marks.
[{name: 'John', id: 123, marks: 98 },{name: 'Wick', id: 115, marks: 75 }]"

Ans: let studentsMarks = studentRecords.filter(student => student.marks > 50);
console.log(studentsMarks);

output:
[ { name: 'John', id: 123, marks: 98 },
{ name: 'Wick', id: 115, marks: 75 } ]
The filter function takes an array and applies a callback function to each element in the array.
The callback function returns true for elements that should be included in the filtered array and false for elements that should be excluded.


3.Retrieve the details of students who scored more than 50 marks and have IDs greater than 120.

Ans:let students120 = studentRecords.filter(student => student.marks > 50 && student.id > 120);
console.log(students120);
output:[ { name: 'John', id: 123, marks: 98 } ]
In the filter function, the callback function checks if both conditions are true for each student. The condition student.marks > 50 checks if the student scored
more than 50 marks, and student.id > 120 checks if the student's ID
is greater than 120. If both conditions are true, the student will be included in the filtered array.


4.Consider the same scenario we have discussed above, but this time we would like to know the sum total of the marks of the students.

Ans:let totalMarks = studentRecords.reduce((sum, student) => sum + student.marks, 0);
console.log(totalMarks);
output:total marks=241
In the reduce function, the callback function takes two parameters: sum and student.
The sum parameter keeps track of the accumulated total, and the student parameter represents
each element in the studentRecords array. The callback function adds the marks of each student to the sum in each iteration.


5.This time we want to get only the names of the students who scored more than 50 marks from the same dataset used above.

Ans:let studentMarks = studentRecords
.filter(student => student.marks > 50)
.map(student => student.name);
console.log(studentMarks);
output:[ 'John', 'Wick' ]
In this code, the filter function is used to filter out the students who scored more than 50 marks,
just like we did in the previous example. The map function is then used to extract only their names from the filtered array.


6.This time we are required to print the sum of marks of the students with id > 120.

Ans:let sumOfMarks = studentRecords
.filter(student => student.id > 120)
.reduce((sum, student) => sum + student.marks, 0);
console.log(sumOfMarks);
output:143
In this code, the filter function is used to filter out the students with an ID greater than 120,
and then the reduce function is used to calculate the sum of their marks,


7.This time we are required to print the total marks of the students with marks greater than 50 after a grace of 15 marks has been added to those students who scored less than 50.

Ans:let totalGrace = studentRecords
.map(student => ({ ...student, marks: student.marks < 50 ? student.marks + 15 : student.marks }))
.filter(student => student.marks > 50)
.reduce((sum, student) => sum + student.marks, 0);
console.log(totalGrace);
output:243
ap function is used to create a new array of students, where the marks of those who scored less than 50 are increased by 15 ( grace marks).
filter function is used to filter out the students who now have marks greater than 50
( applying the grace). Finally, the reduce function is used to calculate the total marks of these filtered students.
